Projects

Handle Time Analysis

This repository features a Jupyter Notebook file named Handle Time Analysis.ipynb. The notebook delves into an analysis of a call center scenario, where the hypothesis under consideration is that phone representatives experience fatigue as the number of calls they handle in a single day increases.

Pet Adoption Import

This repository houses three Python files; Pet Adoption from API, Pet Adoption from CSV, and Pet Adoption from HTML. Each of these files serves the purpose of retrieving pet adoption statistics from the Austin Animal Center using distinct methods.

Predicting Length of Stay for Dogs in Austin Animal Center

This repository features a Jupyter Notebook file named Predicting Length of Stay for Dogs in Austin Animal Center.ipynb. The notebook is dedicated to predicting the length of stay for dogs in the Austin Animal Center, employing linear regression models for accurate estimations.

Weather Application

This repository contains a Python script named Weather Application.py. The purpose of this script is to provide users with real-time weather information based on user input, such as a zip code or city. By leveraging the OpenWeatherMap API, the script retrieves and displays the current temperature for the specified location.
